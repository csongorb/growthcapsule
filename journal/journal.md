# The Growth Capsule?

## 2026-05-10: Rough first notes (Joplin)

reading https://gamestudies.org/2601/articles/mullally I just had an idea for an idle game I have not seen until now (I'm writing this down before I make a research):

an idle game, where I'm investing into the future: basically I can select how much money I want to invest and I can select the duration myself... after the duration I get the money. the longer the duration, the higher the multiplyer.
and I can invest time in a lot of values:

- how fast the earned time loses its value
- the curve with which it looses its value
- the amount of time my investment outcome stays on the same level
- etc.

should be quite doable with AI, isn't it?

money can by your time?

Ok, this is strange, but I immediately feel a good tension. This one-minute-window is great (in my first ideas I thought the decline should be a curve that I can influence, but this is better).
